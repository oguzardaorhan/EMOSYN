# Emosync 

**Emosync** is a real-time facial emotion recognition application that detects and visualizes emotions from webcam input.  
This project is designed as a portfolio piece for software engineering and AI applications, suitable for academic and professional presentation.

---
##  Features

- **Facial Emotion Recognition (FER)** using deep learning
- Detects basic emotions: **Happy, Sad, Angry, Fear, Surprise, Disgust, Neutral**
- Real-time analysis via **webcam**
- **Flask-based web interface** that is responsive and mobile-friendly
- Visual feedback of emotion probabilities on the interface

---
##  Technologies Used

- **Python 3.11**
- **FER** library for emotion detection
- **OpenCV** for real-time video processing
- **Flask** for the web application
- **NumPy, Pandas** for data handling
- **MoviePy** for video processing
- **MTCNN** for accurate face detection
- Fully compatible with **macOS and Linux environments**
